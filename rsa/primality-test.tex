\textbf{Finding Primes for RSA:}        
                \begin{itemize}
\item RSA requires generating huge prime numbers (typically 1024-2048 bits)
\item Process: generate random odd integer of desired bit length, test for primality, if not prime try $n+2$, etc.
\end{itemize}

\textbf{Prime Number Theorem (PNT):}
\begin{itemize}[leftmargin=*,itemsep=0.2em]
\item Discovered by Gauss in 1792/3, proven by Hadamard and de la Vall√©e Poussin in 1896
\item States: $\pi(x) \sim \frac{x}{\ln x}$ as $x \rightarrow \infty$ 
\item Where $\pi(x)$ = number of primes $\leq x$
\end{itemize}

\textbf{Density of Primes:}
\begin{itemize}[leftmargin=*,itemsep=0.2em]
\item Density of primes up to $x$ is approximately $\frac{1}{\ln x}$
\item When searching only odd integers, probability of finding a prime is approximately $\frac{2}{\ln x}$
\end{itemize}

\textbf{Practical Implications:}
\begin{itemize}[leftmargin=*,itemsep=0.2em]
\item For 1024-bit integers, approximately 0.14\% of odd integers are prime
\item Finding a prime typically requires $<$ 1000 primality tests
\end{itemize}

 \textbf{Primality Testing:}
\begin{itemize}[leftmargin=*,itemsep=0.2em]
\item Modern systems use probabilistic primality tests (faster than deterministic tests)
\item Two important tests: Fermat primality test and Miller-Rabin primality test
\item Miller-Rabin is used in real-world applications (builds on Fermat test principles)
\end{itemize}

\textbf{Implementation Considerations:}
\begin{itemize}[leftmargin=*,itemsep=0.2em]
\item Start with random odd integer (least significant bit set to 1)
\item Apply primality test
\item If not prime, try next odd integer ($n+2$)
\item Continue until prime is found
\end{itemize}
